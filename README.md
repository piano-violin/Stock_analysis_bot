## üìä Overview

This Telegram bot analyzes stock prices using three different machine learning models, selects the best performing one, and provides trading recommendations with profit calculations. The bot automatically downloads historical data, trains multiple models, and generates 30-day forecasts.

## ‚ú® Features

- **üìà Multi-model Forecasting**: Implements Random Forest, ARIMA, and LSTM models
- **ü§ñ Automatic Model Selection**: Chooses best model based on MAPE metric
- **üìÖ 30-Day Business Forecast**: Predicts stock prices for next 30 working days
- **üíº Trading Recommendations**: Identifies optimal buy/sell days with profit calculation
- **üìä Interactive Visualization**: Generates historical + forecast plots
- **üìù Comprehensive Logging**: Tracks all user sessions and model performance
- **üîß Error Handling**: Robust error handling for invalid inputs
